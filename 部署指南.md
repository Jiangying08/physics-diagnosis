# 🚀 初三物理诊断系统 · 部署指南

## 📁 文件说明

| 文件 | 用途 | 部署位置 |
|:----|:----|:---------|
| `student.html` | 学生端测试页面 | 部署到 .cn 域名，学生访问 |
| `teacher.html` | 教师端管理后台 | 本地打开即可（无需部署） |

---

## ☁️ 方案一：阿里云 OSS 静态托管（推荐，国内访问最快）

### 第 1 步：购买域名

1. 打开 [阿里云域名注册](https://wanwang.aliyun.com/)
2. 搜索你想要的 `.cn` 域名（如 `wulizhenduan.cn`、`gaotuphy.cn`）
3. 加入清单 → 实名认证 → 付款（约 ¥29/年）

### 第 2 步：开通 OSS

1. 打开 [阿里云 OSS 控制台](https://oss.console.aliyun.com/)
2. 点击「创建 Bucket」
   - **Bucket 名称**：任意英文名（如 `gaotu-physics`）
   - **区域**：选离你最近的城市（如「华东1（杭州）」）
   - **读写权限**：选择 **「公共读」**
3. 点击「确定」

### 第 3 步：上传文件

1. 进入刚创建的 Bucket
2. 点击「上传文件」
3. 将 `student.html` **重命名为 `index.html`** 后上传
4. 上传完成后，点击文件名 → 复制「文件 URL」，试试能不能打开

### 第 4 步：开启静态网站托管

1. 在 Bucket 左侧菜单找到「数据管理」→ **「静态页面」**
2. 点击「设置」
   - **默认首页**：填写 `index.html`
   - **默认 404 页**：填写 `index.html`
3. 点击「保存」
4. 现在通过 Bucket 的「外网域名」就能直接访问了

### 第 5 步：绑定域名

1. 在 Bucket 左侧菜单点击「域名管理」→ **「绑定域名」**
2. 输入你买的 `.cn` 域名（如 `wuli.zhenduan.cn`）
3. 点击「确认」

### 第 6 步：配置 DNS（域名解析）

1. 打开 [阿里云 DNS 控制台](https://dns.console.aliyun.com/)
2. 找到你的域名，点击「解析设置」
3. 点击「添加记录」：
   ```
   记录类型：CNAME
   主机记录：@ （或 www）
   记录值：你的 OSS Bucket 外网域名（如 gaotu-physics.oss-cn-hangzhou.aliyuncs.com）
   ```
4. 点击「确定」，等待 5-10 分钟生效

### ✅ 完成

学生访问 `http://你的域名.cn` 即可做题。

## ☁️ 方案二：腾讯云 COS（与阿里云类似）

1. 在 [腾讯云](https://console.cloud.tencent.com/) 购买域名
2. 开通 COS 对象存储 → 创建 Bucket → 设置「公有读」
3. 上传 `student.html` 为 `index.html`
4. 开启「静态网站」功能
5. 绑定域名 + 配置 DNS（CNAME 指向 COS 域名）

---

## ☁️ 方案三：Vercel（免费，先尝鲜）

> 适合先测试效果，不需要买域名和云服务

1. 打开 [vercel.com](https://vercel.com/) → 用 GitHub 或邮箱注册
2. 登录后点击 **「Add New → Project」**
3. 选择 **「Deploy without Git」**
4. 把 `student.html` **重命名为 `index.html`**
5. 拖拽上传
6. 部署完成后得到一个 `xxx.vercel.app` 的域名
7. 学生直接访问这个域名即可做题
8. 后续可以在 Vercel 的「Domains」设置中绑定你自己买的 `.cn` 域名

## 🔄 教师端使用

`teacher.html` **不需要部署**，你在自己电脑上打开就行：

1. 学生完成测试后，复制 **分享码** 发给你（微信/私信）
2. 你打开 `teacher.html` → 点「导入学生」→ 粘贴分享码
3. 学生自动出现在列表中，点击可查看完整教师版报告

**数据导出备份：**
- 教师端支持「导出全部」为 JSON 文件
- 换电脑时点「导入备份」即可恢复所有学生数据

---

## 💡 小建议

1. **域名选择**：建议用简短好记的，如 `phy-test.cn`、`wuli.cn` 之类的
2. **https 证书**：阿里云/腾讯云 OSS 可以免费开启 HTTPS（SSL 证书），建议开启，更安全
3. **微信分享**：部署好后，把学生端域名分享到微信班级群，学生点击就能直接打开做题
